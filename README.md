
# Генератор Логотипов с использованием Yandex-Art API

Этот проект представляет собой веб-приложение для генерации логотипов с использованием Yandex-Art API. Пользователь вводит форму, стиль и описание логотипа, а система генерирует изображение на основе этих данных.

## Описание

Приложение использует Flask для создания веб-интерфейса и Yandex-Art API для генерации изображений. Оно позволяет пользователям:
- Вводить форму, стиль и описание логотипа через веб-форму.
- Отправлять запрос на генерацию изображения через Yandex-Art API.
- Получать сгенерированное изображение и сохранять его на сервере.

## Установка

### 1. Клонирование репозитория

Склонируйте репозиторий на свою машину:

```bash
git clone https://github.com/MasterVVK/GenLogo.git
```

### 2. Создание виртуального окружения

Создайте и активируйте виртуальное окружение:

#### Для Windows:
```bash
python -m venv venv
venv\Scripts\activate
```

#### Для macOS/Linux:
```bash
python3 -m venv venv
source venv/bin/activate
```

### 3. Установка зависимостей

Установите все зависимости, указанные в `requirements.txt`:

```bash
pip install -r requirements.txt
```

### 4. Настройка переменных окружения

Создайте файл `.env` в корне проекта и добавьте в него свои переменные:

```bash
OAUTH_TOKEN=ваш_oauth_токен
FOLDER_ID=ваш_folder_id
IMAGES_PATH=static/images
```

- `OAUTH_TOKEN`: Ваш OAuth-токен от Yandex Cloud.
- `FOLDER_ID`: Идентификатор папки в Yandex Cloud.
- `IMAGES_PATH`: Путь для сохранения изображений.

### 5. Запуск приложения

Запустите приложение:

```bash
python app.py
```

Приложение будет доступно по адресу `http://127.0.0.1:5000`.

## Структура проекта

- **config.py**: Конфигурационный файл для загрузки переменных окружения.
- **app.py**: Основной файл Flask-приложения.
- **logo_gen.py**: Логика генерации логотипа с использованием Yandex-Art API.
- **services/yandex_service.py**: Сервис для работы с Yandex-Art API.
- **templates/index.html**: HTML-шаблон для ввода данных.
- **static/images**: Папка для сохранения сгенерированных изображений.

## Пример работы

После запуска приложения откройте браузер и перейдите по адресу:

```
http://127.0.0.1:5000
```

Заполните форму с параметрами для логотипа (форма, стиль, описание) и нажмите "Сгенерировать логотип". Через несколько секунд вы получите сгенерированное изображение.
